

## Project Dependencies:
```
* Body-Parser

* Bookshelf

* Express

* Knex

* Morgan

* sqlite3

* Mustache

* Handlebars

* Handlebars-Express
```
 
## Run Code:
Make sure to use: `node v11.10.0 (npm v6.7.0)`. To get it, use: `nvm install stable`
```
*  npm install
*  nvm install stable
*  knex migrate:latest --env development
*  knex seed:run --env development
*  npm start
```
